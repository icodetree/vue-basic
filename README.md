# vue-basic

## basic setup

-   버전확인

```
vue --version
```

-   vue 설치

```
// window
npm install -g vue

// macOS
sudo npm install -g vue
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
